# Changelog

All notable changes to Urban Explorer will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Firebase App Distribution integration for APK releases
- Firebase Hosting for web deployments
- Automated release workflow with GitHub Actions

## [1.0.0] - 2025-12-05

### Added
- Initial release of Urban Explorer
- Interactive map with OpenStreetMap (Leaflet)
- Tab-based navigation (Explore, Search, Favorites, Profile)
- Location markers with category filtering
- User geolocation with "center on me" functionality
- Search functionality with real-time results
- Responsive design for mobile and web
- Clean Architecture implementation
- Supabase integration for backend services

### Technical
- Ionic 8 + Angular 20 + Capacitor 6
- Standalone components architecture
- TypeScript strict mode
- ESLint + Prettier configuration
- CI/CD pipeline with GitHub Actions
- 80% test coverage threshold
