# Urban Explorer Frontend

Mobile application for urban exploration built with Ionic 8 + Capacitor 6 + Angular 18.

## Tech Stack

- **Framework**: Ionic 8
- **Runtime**: Capacitor 6
- **UI Framework**: Angular 18
- **State Management**: NgRx (signals)
- **Testing**: Jest + Cypress

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
ionic serve

# Run on iOS
ionic cap run ios

# Run on Android
ionic cap run android
```
